= Quick Start

Apply the `com.citi.helm` plugin to your Gradle project:

[source,groovy,role="primary",subs="+attributes"]
.Groovy
----
plugins {
    id 'com.citi.helm' version '{project-version}'
}
----

[source,kotlin,role="secondary",subs="+attributes"]
.Kotlin
----
plugins {
    id("com.citi.helm") version "{project-version}"
}
----
